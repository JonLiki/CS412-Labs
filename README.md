# CS412: Artificial Intelligence and Machine Learning

### Google Colab Lab Repository

This repository contains my laboratory exercises and assignments for the **CS412** course. All projects are developed using **Python** within the **Googleolab** environment.

## 📂 Repository Content

* **`Labs/`**: Weekly laboratory exercises exploring AI algorithms.

* **`Assignments/`**: Major course assignments and programming tasks.

* **`Data/`**: Sample datasets used for training and testing models.

## 🚀 How to Run the Notebooks

1. Navigate to the desired folder (e.g., `Labs/`).

2. Click on any `.ipynb` notebook file.

3. Click the **"Open in Colab"** button at the top of the file to launch the interactive environment.

## 🛠️ Tech Stack

* **Environment:** Google Colab / Jupyter Notebooks

* **Language:** Python 3.x

* **Libraries:** NumPy, Pandas, Matplotlib, Scikit-Learn
